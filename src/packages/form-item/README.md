# Form Item

## Props

| Name          | Type     | Default  | Description |
|---------------|----------|----------|-------------|
| tag           | string   | "div"    |             |
| label         | string   | ""       |             |
| hint          | string   | ""       |             |
| for           | string   | ""       |             |
| hasError      | boolean  | false    |             |
| errorMessage  | string   | ""       |             |

## Slots

| Name    | Description                                  |
|---------|----------------------------------------------|
| default | The default Vue slot.                        |
| label   | Content to place inside the <label> element. |
| hint    | Content to place after default slot.         |
